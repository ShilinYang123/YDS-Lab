## 一、项目定位

开发一款**本地运行、轻量高效**的桌面级视频去水印工具，服务于：
- 内容创作者
- 视频剪辑师
- 短视频运营者

**核心价值**：  
✅ 保护用户隐私（无需上传）  
✅ 高质量修复（AI + OpenCV）  
✅ 支持离线使用（端侧AI）

---

## 二、技术栈

| 模块 | 技术选型 |
|------|----------|
| 前端界面 | Tauri + React |
| 后端逻辑 | Python（通过 Tauri 调用） |
| 核心算法 | FFmpeg + OpenCV（阶段1） → 轻量U-Net（阶段2） |
| AI 模型 | ONNX Runtime（<1GB） |
| 依赖工具 | FFmpeg, Shimmy, Git, Python 3.14+ |
