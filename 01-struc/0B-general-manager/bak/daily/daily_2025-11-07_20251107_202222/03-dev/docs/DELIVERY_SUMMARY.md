## 🎯 使用指南

### 快速启动
1. 进入项目根目录: `cd S:\YDS-Lab`
2. 运行启动脚本: `tools\scripts\start_production.bat`
3. 访问系统管理界面
4. 查看用户指南: `Struc\GeneralOffice\Docs\UserGuides\trae_platform_user_guide.md`

### 系统管理
- **监控**: 查看根目录 `logs\` 下的日志文件
- **配置**: 修改 `config\` 目录下的配置文件
- **备份**: 使用 `tools\backup\daily_snapshot.py` 进行备份
- **故障排除**: 参考 `Documentation\Troubleshooting\troubleshooting_guide.md`
