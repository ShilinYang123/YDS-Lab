# AI开发团队可视化会议室

一个基于Web的可视化会议室系统，支持多Agent协作会议，具有语音识别、白板功能和演示文稿展示。

## 功能特点

### 核心功能
- **可视化会议室**: 长桌式布局，直观的Agent位置显示
- **多Agent参会**: 6个不同角色的AI开发团队成员
- **语音识别**: 每个Agent有独特的语音特征
- **实时聊天**: 会议聊天功能，支持消息记录
- **白板功能**: 实时协作白板，支持多人编辑
- **演示文稿**: 支持PPT/方案展示
- **移动端适配**: 响应式设计，支持手机参会

### Agent角色
1. **前端开发工程师**: React、TypeScript专家
2. **后端开发工程师**: FastAPI、Python专家  
3. **系统架构师**: 微服务、云原生专家
4. **DevOps工程师**: CI/CD、容器化专家
5. **项目经理**: 项目管理、团队协调
6. **测试工程师**: 自动化测试、质量保证

### 技术特性
- **实时通信**: Socket.IO实现实时消息传递
- **语音合成**: Web Speech API支持文本转语音
- **响应式设计**: Bootstrap 4 + 自定义CSS
- **模块化架构**: 易于扩展和维护
- **数据持久化**: 会议状态和聊天记录保存

## 快速开始

### 环境要求
- Node.js 14+
- npm 6+
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 安装部署

1. **克隆项目**
```bash
git clone <repository-url>
cd ai-meeting-room
```

2. **安装依赖**
```bash
npm install
```

3. **启动服务**
```bash
# 方式1: 使用Node.js直接启动
node index.js

# 方式2: 使用启动脚本
chmod +x start.sh
./start.sh
```

4. **访问系统**
打开浏览器访问: http://localhost:3000

## 使用指南

### 会议控制
- **开始会议**: 点击"开始会议"按钮
- **结束会议**: 点击"结束会议"按钮
- **录音功能**: 点击"录音"按钮开始/停止录音

### Agent操作
- **选择发言人**: 点击Agent头像或列表项
- **发送消息**: 在聊天框输入消息并发送
- **语音播放**: Agent发言时会自动播放语音

### 白板使用
- **编辑内容**: 点击白板区域直接编辑
- **实时同步**: 所有参会者都能看到最新内容

### 演示文稿
- **上传PPT**: 支持HTML格式的演示文稿
- **全屏展示**: 点击展示按钮全屏查看
- **多人观看**: 所有参会者同步观看

## 扩展开发

### Agent扩展
可以通过修改`index.js`中的`agents`数组添加新的Agent：

```javascript
const agents = [
    {
        id: 'new-agent',
        name: '新Agent名称',
        role: '角色类型',
        avatar: 'emoji头像',
        color: '主题颜色',
        voice: '语音类型',
        status: 'online',
        expertise: ['技能1', '技能2']
    }
];
```

### 语音扩展
可以添加新的语音类型，需要：

1. 在`public/audio`目录添加语音文件
2. 在Agent配置中指定新的voice类型
3. 在前端代码中添加对应的语音参数

### 功能扩展
系统采用模块化设计，可以方便地添加新功能：

1. **新页面**: 在`views`目录添加EJS模板
2. **新API**: 在`index.js`中添加路由
3. **新组件**: 在`public/js`中添加JavaScript组件

## 故障排除

### 常见问题

1. **服务启动失败**
   - 检查Node.js版本
   - 检查端口是否被占用
   - 检查依赖是否安装完整

2. **WebSocket连接失败**
   - 检查网络连接
   - 检查防火墙设置
   - 检查服务器地址是否正确

3. **语音播放失败**
   - 检查浏览器权限
   - 检查系统音量设置
   - 确认浏览器支持Web Speech API

4. **白板同步问题**
   - 检查网络连接
   - 刷新页面重试
   - 检查Socket.IO连接状态
