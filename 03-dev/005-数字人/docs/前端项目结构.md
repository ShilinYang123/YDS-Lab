# 前端项目结构文档

## 概述
前端项目基于 React 18 + TypeScript 构建，采用现代化的前端开发架构，确保代码的可维护性和可扩展性。

## 技术栈

### 核心依赖
- **React 18**: 前端框架
- **TypeScript**: 类型安全
- **Vite**: 构建工具
- **React Router v6**: 路由管理
- **Axios**: HTTP客户端
- **Zustand**: 状态管理

### UI组件库
- **Ant Design**: 企业级UI组件库
- **Tailwind CSS**: 实用优先的CSS框架
- **Framer Motion**: 动画库

### 开发工具
- **ESLint**: 代码检查
- **Prettier**: 代码格式化
- **Husky**: Git Hooks
- **lint-staged**: 暂存文件检查

## 开发规范

### 组件命名
- 组件文件使用 PascalCase 命名
- 组件文件夹与组件同名
- 样式文件使用 CSS Modules

### 代码组织
- 每个组件包含：
  - `index.tsx` - 主组件文件
  - `styles.module.css` - 组件样式
  - `types.ts` - 组件类型定义
  - `utils.ts` - 组件工具函数

### 状态管理
- 全局状态使用 Zustand
- 组件状态使用 useState/useReducer
- 避免过度使用全局状态

### API调用
- 所有API调用通过 services 层
- 统一错误处理
- 请求/响应拦截器

## 环境配置

### 开发环境
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建项目
npm run build

# 运行测试
npm run test
```

### 环境变量
```env
VITE_API_URL=http://localhost:3001
VITE_WS_URL=ws://localhost:3001
VITE_ENV=development
```
